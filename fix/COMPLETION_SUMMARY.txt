================================================================================
KEY SOLUTION DETAILS
================================================================================

PROBLEM SOLVED:
  Firefox Mobile and old Android browsers couldn't use responsive-scale-mixins
  because CSS variables weren't supported.

SOLUTION IMPLEMENTED:
  Automatic fallback generation during SCSS compilation creates two CSS
  declarations per property:
  1. Static fallback value (for old browsers)
  2. Modern calc() expression (for new browsers)
  
  Browsers automatically use whichever they understand.
